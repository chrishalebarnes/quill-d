    Class Mapper

    Maps parameters, and results to a class specified by T.

    Inherits from

    • Object (base class)

    Methods

    Name Description
    map Maps each property of T to a column in a ResultSet. If T is a class, it will new it up and map all of the public fields. If T is a primitive type it will assume the result of the query boils down to a single value and returns that.
    map Maps a class into a PreparedStatement
    mapArray Maps an array of models where the ResultSet is expected to have multiple rows
    mapOne Maps a single T where the ResultSet is expected to have only one row.
    mapType Sets a parameter on the PreparedStatement if it is supported
    mapType Gets a value out of the current row if it is supported.
